# Dashboard

Dashboard de Starwars utilizando a SWAPI.

Dashboard criada com o intuito de estudar requisições API, utilizando uma API pública de StarWars.

<div align="left">
<img src="https://user-images.githubusercontent.com/84355579/159811385-93f1bc7a-30f9-49a4-81a0-6356ba60ea3f.jpg" width="450px" />
</div>

## Funcionalidades

- Requisições GET
- Tabela com informações dos 6 primeiros filmes
- Gráfico utilizando Google Charts
